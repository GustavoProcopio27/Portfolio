# Portfólio Pessoal

Este repositório contém o código-fonte do meu **site de portfólio pessoal**, desenvolvido com o objetivo de apresentar meus **projetos**, **experiência**, **habilidades** e informações de contato de forma clara e acessível.



## Tecnologias Utilizadas
- Vite.
- React.  
- Tailwind CSS.
- TypeScript.

## Bibliotecas Utilizadas
- **i18next** : Utilizada para tradução da página, tornando-a bilíngue.
- **react-icons** : Utilizada para adição de ícones de navegação e de estilização.
- **react-router-dom** : Utilizado para roteamento de páginas
- **react-country-flag** : Para os ícones das bandeiras do Brasil e dos Estados Unidos para o botão de tradução.
- **gh-pages** : Para deploy no github, em uma url com github.io.
