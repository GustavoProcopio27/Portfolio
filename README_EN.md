## Features
- Bilingual website (Portuguese / English)
- Fully responsive layout
- Page routing with React Router
- Modern interface styled with Tailwind CSS
- Projects section with visual highlights

## Libraries Used:
- **i18next**: Used for page translation, making the website bilingual.
- **react-icons**: Used for navigation and styling icons.
- **react-router-dom**: Used for client-side routing.
- **react-country-flag**: Used to display the Brazilian and United States flags for the language switcher.
- **gh-pages**: Used to deploy the project on GitHub Pages with a github.io URL.
